## Instrucciones para el asistente

- Al iniciar cualquier conversación, lee `project_active_work.md` para saber qué hay en curso.
- Si el usuario pregunta "por dónde íbamos", "vamos", "continuamos" o similar, consulta el trabajo activo y responde con el estado actual y el siguiente paso.
- Nunca implementes código sin un spec consensuado que el usuario haya aprobado explícitamente.
- Cuando se complete un punto de un plan, actualiza su estado en el plan y actualiza `project_active_work.md` con el siguiente paso y la fecha.
- Cuando se complete un plan entero, retira la referencia de trabajo activo y archívalo.
- Si no hay trabajo activo, dilo y pregunta qué hacer.
